notas consola

  1. cd: moverse entre carpetas
  2. cd: ..: moverse a carpeta madre
  3. cd: "nombre carpeta" moverse a carpeta hija
  4. cd: dir: muestra los directorios hijos
  5. cd: mkdir: crear directorio
  6. cd: cls: limpiar consola
  7. cd: vs -v: version visual studio
  8. cd: git -v: version git
  9. cd: node -v: version node
  10. añadir mas...

notas git

  • git init: inicializar repositorio git
  • git status: comprobar estado del repositorio
  • git branch: listado ramas
  • git checkout [nombre rama] : crear rama
  • git checkout -b [nombre rama] crear rama y cambiarse a ella
  • git branch [ nombre rama]: crear rama
  • git add . : añadir al stage todo
  • git commit -m "mensaje": perpetrar los cambios para que haya seguimiento
  • git config --global user.name "FIRST_NAME LAST_NAME": configurar usar name
  • git config -- global user.email "email#email.com": configuar email
